Descripcion

Debido a lo tedioso que puede llegar a ser el sobrescribir una aplicación de un lenguaje a otro se le solicita que, como estudiante de sistemas, realice un traductor de lenguaje de programación C# a Python, el cual contiene algunas de las sentencias básicas del lenguaje como lo son declaración y asignación de variables, sentencias de retorno, sentencias de condición y repetición, métodos, funciones entre otros; Del cual se espera que genere la salida del lenguaje python así como el resultado final de texto tipo HTML que contendrá el archivo a traducir.

Herramientas utilizadas:

  • JavaScript(backend)
  • Vue(Frontend)
  • Jison

Paquetes y Clases:

  • Analizador se utilizo la herramienta de “JISON” .

  • Analizador: Esta carpeta tiene los diferentes archivos de las estructuras utilizadas para la captura de datos de todas las instrucciones.
    Se utilizo por cada instruccion un clase para poder obtener los datos de las instrucciones utilizadas en este proyecto. Cada clase tiene un metodo operar que nos sirve para poder enviar los datos a la tabla de simbolos y un metodo Traducir que traduce cada instruccion.
    Estos son algunas de las clases utilizadas:

    • Declaracion.js
    • Imprimir.js

  • Instruccion.js: Esta clase obtenia todas las instrucciones y concatenaba con ayuda del metodo prueba todas las traducciones.
